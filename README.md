WordCloud Generator - 纯前端词云生成工具
一个轻量高效的纯前端词云生成器，无需后端依赖，本地即可快速生成个性化词云图，满足多样化的可视化需求。

✨ 核心功能
自定义形状上传：支持上传图片作为词云轮廓（如 logo、图形等），让词云更具辨识度
灵活样式配置：
内置主题配色方案，一键切换风格
支持自定义文字颜色、字体、字号范围及旋转角度
透明轮廓实时预览，直观调整词云布局
多格式导出：生成的词云可导出为 PNG（高清图片）或 SVG（矢量格式，无损缩放）

🚀 快速使用
下载仓库完整文件（Code → Download ZIP）
解压后，直接打开根目录的 index.html 文件
上传文本 / 选择形状 → 调整样式 → 生成并导出词云
